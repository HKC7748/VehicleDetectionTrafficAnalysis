# 车辆检测与交通流量分析系统

## 使用说明

### 1. 环境配置

打开终端，依次执行以下命令：

```bash
# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate

# Mac/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

### 3. 使用流程

#### 第一步：配置检测参数

在打开的窗口中，切换到 **配置** 标签页：

1. **导入配置**：点击"导入配置"，选择已有的 JSON 配置文件
2. **选择模型**：点击 YOLO 模型路径旁的"浏览"，选择 `.pt` 模型文件
3. **选择视频**：点击视频源旁的"浏览"，选择待分析的视频文件
4. **设置车辆类型**：在"车辆类型"表格中添加或修改车辆尺寸参数
5. **设置透视/平行点**：在"透视/平行点"区域添加对应的坐标点
6. **添加车道**：点击"添加"按钮创建车道，填写车道名称、检测线点和边界点


#### 第二步：开始检测

切换到 **检测** 标签页：

1. 点击"开始检测"按钮
2. 等待检测完成，查看实时输出日志
3. 检测完成后自动显示统计结果

#### 第三步：查看结果

切换到 **可视化** 标签页：

1. 拖动帧选择滑块，查看不同时刻的检测画面和车辆轨迹
2. 点击"一键导出所有数据"，选择导出目录，将所有统计数据保存为 CSV 文件
